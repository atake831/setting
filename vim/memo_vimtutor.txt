i : 挿入モード
R : 置換モード
v : ビジュアルモード

移動
    0 : 行頭
    e : １単語
    G : 最下行
    gg : ファイルの先頭
    数値 + G : (数値)行に移動
    ctrl + o : 元のカーソル位置に戻る
    ctrl + i : ctrl + o の前方向

オペレータ　＋　（カウント）　＋　モーション

オペレータ
    d : 削除オペレータ
    c : 変更オペレータ
    y : コピーオペレータ

モーション
    w : カーソル位置から空白を含む単語の末尾まで。
    e : カーソル位置から空白を含まない単語の末尾まで。
    $ : カーソル位置から行末まで。

削除
    x : １文字削除
    dw : 1単語の削除
    d$ : 行の末尾までを削除
    dd : 1行全体の削除
    v + 選択 + d : 選択された部分の削除

テキスト追加
    i : カーソル位置にテキスト追加
    A : 行末にテキストの追加
    a : カーソルの次の位置からテキストを追加

u : undo
U : 行延滞のundo
ctrl + r : redo

o : カーソルの下の行を開き、挿入モードになる
O : カーソルの上の行を開き、挿入モードになる

コピーとペースト
    p : 張り付け
    y : 1行コピー
        v + 選択 + y : 選択部分のコピー

変更
    r : １文字の置き換え
    cw : 単語の一部、もしくは全体の変更(w e $を使用可能)

ファイル情報
    ctrl + g : カーソル位置情報とファイルの状態を表示

検索
    / : 検索コマンド
        n : もう一度検索
        N : 逆方向に検索
    ? : 逆方向に検索
    % : 対応する括弧を検索

置換
    :s/old/new : その行で最初に見つかったoldをnewに置換
    :s/old/new/g : 行全体のoldをnewに置換
    :%s/old/new/g : ファイル全体の置換
    :%s/old/new/gc : ファイル全体を確認を取りながら置換

    R : 置換モードへ
        １文字以上を置き換える
        置換モードは全てのタイプされた文字は既存の文字を削除（置換）する
    
外部コマンド
    :! + コマンド : 外部シェルコマンドを実行

:w : 保存
v + 選択 + :w ファイル名 : 選択部分をファイルに書き込む
:r ファイル名 : ファイルの中身を読み込んで挿入
    :r !ls などの外部コマンドも使用可能

バーチャル
    v : 選択
        + d : 選択部分の削除
        
:set オプション : オプションを設定
    :set no+オプション : オプションを無効にする
